1.本客户端实现了员工的姓名、年龄、电话，账号、密码、工号、工资、部门等信息的注册管理。

2.本项目设置了一个超级管理员，登录的时候会自动识别出是不是管理员在登陆，（管理员可修改用户的薪资部门及所有信息）
	管理员账号为：777   密码：777

3.实现原理： （1）客户端启动后 填充服务器信息结构体，请求连接服务器，为了方便运行连接，服务器，客户端ip“0.0.0.0”（本机所有网卡）
              	     （2）连接请求建立好后，并根据提示输入相应的操作码，及操作所需信息，写入信息传递结构体，发送给服务器。
4、操作码
	#define ENTER        100              //登录                   
	#define ENTERWIN     1001        //登录成功
	#define QUIT         101 	    //退出                  
	#define REGISTER     102 	    //注册
	#define REGISTERWIN  1021     //注册成功
	#define USERADD      103         //注册填充信息             
	#define USERADDWIN   1031  //填充信息成功                
	#define ERR          -1                //失败通用标识              
	#define LOOKMES      104       //查询个人信息
	#define ALTERMES     105       //修改个人信息                
	#define ALTERMESWIN  1051       //修改个人信息成功
	#define ALTERCODE    106           //修改密码            
	#define ALTERCODEWIN 1061    //修改密码成功
	#define ROOT         107              //超级用户功能                   
	#define ROOTWIN      1071       //超级功能成功              
	#define ROOTDL       108          //超级用户删除用户信息
	#define ROOTDLWIN    1081      //超级用户删除用户信息成功             
	#define ROOTALT      109               //超级用户修改功能                  
	#define ROOTALTWIN   1091       //超级用户修改功能成功

5、信息传递结构体 
struct  cli_users{  
         int opt;              //操作数
         char name[20];        //姓名
         int  age;             //年龄
         int  account;         //账号  
         int  code;            //密码
         int  jobnum;          //工号
         int  salary;  	      //工资
         char department[20];  //部门
         int phone;   //电话
}user;

1.1：登录，填充账号，密码，判断是否为超级管理员，并填入对应操作码，发送给服务器，并接收服务器发送的信息，
		通过操作码判断此时状态（登录成功或登陆失败）
		 
	
1.2：退出，主动断开连接，（return0），断开之前给服务器发送断开操作码后直接return 0；

1.3：注册，填充账号，密码，并填入对应操作码，发送给服务器，并接收服务器发送的信息，
		通过操作码判断此时状态（注册成功或注册失败）

1.4  查看个人信息：登陆成功后才普通用户可执行的操作，步骤：填入对应操作码，发送给服务器。
		接收到服务器信息后打印个人姓名年龄工号电话等信息

1.5  修改个人信息：登陆成功后才普通用户可执行的操作，步骤：填入对应操作码，填入个人信息，发送给服务器。
		接收到服务器信息后根据操作码成功打印修改成功

1.6  修改密码：：登陆成功后才普通用户可执行的操作，步骤：填入对应操作码，填入原密码，与登录密码比对，不一样打印提示信息	
		一样就输入新密码发送给服务器。接收到服务器信息后根据操作码判断是否修改成功，成功打印修改成功重新登录

1.7  超级用户功能类似，与上述类似，打印的提示有所变化，以及权限更大，，可登录账号777，密码777体验
